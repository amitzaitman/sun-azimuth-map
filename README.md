# זווית השמש על מפה טופוגרפית

דף אינטרנט אינטראקטיבי שמציג את **אזימוט** ו**גובה** השמש ביחס למיקום ולשעה, על גבי מפה טופוגרפית פתוחה (OpenTopoMap).

## תכונות
- מפה טופוגרפית מפורטת (OpenTopoMap)
- קבלת מיקום אוטומטית מהמכשיר (GPS) או בחירה ידנית (לחיצה על המפה / הזנת קואורדינטות)
- בחירת תאריך ושעה, או כפתור "עכשיו"
- הצגת:
  - גובה השמש מעל האופק (°)
  - אזימוט (כיוון מהצפון + כיוון קרדינלי בעברית)
  - זמני זריחה, שקיעה, צהריים סולאריים ושעת זהב
- קו כתום = כיוון השמש
- קו אפור מקווקו = כיוון הצל

## הרצה מקומית
פשוט פתח את `index.html` בדפדפן (מומלץ דרך שרת מקומי פשוט אם יש בעיות CORS עם tiles).

## GitHub Pages
1. לך ל־**Settings → Pages** של הריפו הזה
2. Source: **Deploy from a branch** → בחר `main` ו־`/ (root)`
3. אחרי דקה-שתיים האתר יהיה זמין בכתובת:
   **https://amitzaitman.github.io/sun-azimuth-map/**

## טכנולוגיות
- [Leaflet](https://leafletjs.com/) – מפה
- [OpenTopoMap](https://opentopomap.org/) – שכבת טופוגרפיה
- [SunCalc](https://github.com/mourner/suncalc) – חישוב מיקום השמש
