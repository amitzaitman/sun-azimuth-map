# זווית השמש + הסתרה על ידי הרים

דף אינטרנט אינטראקטיבי שמציג את **אזימוט** ו**גובה** השמש ביחס למיקום ולשעה, על גבי מפה טופוגרפית, **ובודק האם הרים מסתירים את השמש**.

## תכונות
- מפה טופוגרפית מפורטת (OpenTopoMap)
- קבלת מיקום אוטומטית מהמכשיר (GPS) או בחירה ידנית (לחיצה / גרירה / הזנה)
- בחירת תאריך ושעה + כפתור "עכשיו"
- הצגת גובה השמש ואזימוט (עם כיוון בעברית)
- **בדיקת הסתרה על ידי תוואי שטח** (SRTM 30מ׳):
  - דגימת נקודות לאורך כיוון השמש
  - חישוב זווית גובה של הרכסים (כולל תיקון עקמומיות כדור הארץ)
  - סימון נקודת החסימה על המפה אם קיימת
- אפשרות להגדיר גובה עין / גג (מטרים מעל הקרקע)
- זמני זריחה, שקיעה, צהריים סולאריים ושעת זהב

## GitHub Pages
1. Settings → Pages → Deploy from a branch → `main` / root
2. האתר: **https://amitzaitman.github.io/sun-azimuth-map/**

## הערות
- בדיקת ההסתרה משתמשת ב-API ציבורי של OpenTopoData + proxy (כי ל-API הציבורי אין CORS). לעיתים השרת עמוס – פשוט נסה שוב.
- רזולוציית SRTM ~30 מ׳ – טובה להרים גדולים, פחות מדויקת לבניינים או עצים.
